# Portal PDF

A lightweight PDF reader (and later editor, too)

---

## How to Use

### 1. Requirements

Make sure you have **Python 3.8 or newer** installed. Check your version by running:

```bash
python --version
```

### 2. Install Dependencies

```bash
pip install pyside6 pymupdf
```

### 3. Clone The Repository

```bash
git clone https://github.com/Ruzegon/portal-pdf.git
cd portal-pdf
```

### 4. Run

```bash
python main.py
```

---

## Features

- Open PDF
- Zoom
- Search
- Print
